
# 💬 BARTA Desktop

> **Fast — Minimal — Secure. Chat with your people in style.**

BARTA Desktop is a clean, dark-themed responsive web messaging application built as part of the **2nd Semester Project-Based Learning (PBL-2)** curriculum. It focuses on offering a highly minimalist and sleek user experience for seamless communication.

![BARTA Desktop Landing Page](image_cc0e5a.png)

---

## ✨ Features
* **Minimalist Landing Page:** A beautiful, dark-mode welcome portal featuring a centered call-to-action layout (`index.html`).
* **User Authentication:** Clean and secure UI layout for accessing the chat application (`login.html`).
* **Interactive Chat Interface:** An intuitive message view window equipped with custom typography and streamlined layout spaces (`chat.html` / `chat_box.html`).
* **Fluid Stylesheet:** Modern vanilla CSS focusing on tight typography, flex-centering, and elegant interactive elements like the brand action buttons (`style.css`).
